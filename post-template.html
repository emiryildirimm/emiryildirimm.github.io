Comprehensive Network Scanning with Nmap

March 12, 2026 5 min read Network Security

During my cybersecurity internship, one of the most fundamental tools I use daily is nmap (Network Mapper). It is essential for network discovery and security auditing.

Basic Scanning Techniques

Before diving into complex scripts, it is crucial to understand the basics. A simple ping scan allows us to see which hosts are alive on a subnet without sending any packets to ports.

# Perform a ping scan on a /24 subnet
nmap -sn 192.168.1.0/24

Advanced Service Detection

Once active hosts are identified, the next step is determining what services and versions are running on open ports. This helps in mapping vulnerabilities to specific service versions.

  • -sV: Probes open ports to determine service/version info.
  • -sC: Runs default Nmap scripts for advanced discovery.
  • -p-: Scans all 65535 ports instead of the default 1000.
# Scan all ports with version detection and default scripts
nmap -p- -sV -sC 10.10.10.5

Conclusion

Understanding the exact flags to use in different scenarios saves time and reduces noise on the network. In the next post, I will cover how to use Burp Suite to analyze the web applications hosted on the ports we just discovered.
